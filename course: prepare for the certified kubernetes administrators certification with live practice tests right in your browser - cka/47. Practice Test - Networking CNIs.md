Which CNI plugin is currently installed on the cluster?
```bash
ls /opt/cni/bin/
```

Flannel CNI plugin suppot network policy?
- No

Calico CNI supports network policy?
- YES

The Flannel CNI does not support NetworkPolicies.

Delete Flannel CNI.
-  Is the Flannel controller pod running?
```bash
kubectl delete ds  kube-flannel-ds -n kube-flannel 
```

- Is the Flannel configuration ConfigMap present?
```bash
kubectl delete cm kube-flannel-cfg -n kube-flannel
```

- Is the Flannel configuration file available?
```bash
rm -rf  /etc/cni/net.d/10-flannel.conflist
```
