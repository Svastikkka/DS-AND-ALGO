Create a static pod named static-busybox that uses the busybox image , run in the default namespace and the command sleep 1000
```bash
kubectl run  static-busybox  --image=busybox --namespace=default  --command -- sleep 1000
```

```yaml
apiVersion: v1
kind: Pod
metadata:
  generation: 1
  labels:
    run: static-busybox
  name: static-busybox
  namespace: default
spec:
  containers:
  - command:
    - sleep
    - "1000"
    image: busybox:1.28.4
    imagePullPolicy: Always
    name: static-busybox
    resources: {}
```
**Note**: Check kubelet logs to verify what is happening `journalctl -u kubelet -f`
