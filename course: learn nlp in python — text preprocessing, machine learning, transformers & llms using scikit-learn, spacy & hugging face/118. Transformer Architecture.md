### Transformer Architecture

```
Raw Text -> Embedding Layer -> Attention Layer -> FNN Layer -> Prediction
```
🧠 Transformer Architecture (Step by Step)

The original Transformer from “Attention Is All You Need” (2017) has two main parts:
- Encoder – Reads the input sequence (e.g., English sentence).
- Decoder – Generates the output sequence (e.g., French translation).

Most modern LLMs (like GPT) use only the decoder, but understanding the full architecture is very helpful.
