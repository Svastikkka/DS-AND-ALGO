
1️⃣ Input & Embedding Layer

- Input text → tokenized into integers ([101, 3562, 2003, ...])
- Tokens → mapped to embeddings (vectors of size d_model, e.g. 512 or 768)
- Positional Encoding is added because transformers process all tokens in parallel (they don’t inherently know order).

🔑 Key Formula:
```
Input Representation = Token Embedding + Positional Encoding
```

2️⃣ Encoder Block (Repeated N times)

Each encoder block has:

1. Multi-Head Self-Attention
    - Every token “looks at” every other token in the input sequence.
    - Produces a weighted sum of all tokens → focuses on relevant words.
    - Attention Formula:

        $$
        \text{Attention}(Q, K, V) = \text{softmax}\!\left(\frac{QK^{T}}{\sqrt{d_k}}\right)V
        $$

        Where:
        - Q = Queries
        - K = Keys
        - V = Values
    - Multi-head means we compute several attentions in parallel and concatenate results — allowing the model to capture different relationships (syntax, meaning, etc.).
2. Add & Norm
- Residual connection: Output of attention is added back to the input.
- Layer normalization: Keeps activations stable during training.

3. Feed-Forward Network (FFN)
- Two linear layers with ReLU (or GELU) activation:
- Applied independently to each token vector.

$$
\text{FFN}(x) = \max(0, xW_1 + b_1)W_2 + b_2
$$

Where
- $x \in \mathbb{R}^{d_{\text{model}}}$ = input vector  
- $W_1 \in \mathbb{R}^{d_{\text{model}} \times d_{\text{ff}}}$, $b_1 \in \mathbb{R}^{d_{\text{ff}}}$ = first layer weights & bias  
- $W_2 \in \mathbb{R}^{d_{\text{ff}} \times d_{\text{model}}}$, $b_2 \in \mathbb{R}^{d_{\text{model}}}$ = second layer weights & bias  
- $\sigma(\cdot)$ = activation function (ReLU or GELU)

4. Add & Norm (Again)
- Another residual connection + normalization after FFN.

3️⃣ Decoder Block (Repeated N times)

- The decoder is similar to the encoder but has one extra step:
- Masked Self-Attention – prevents looking at future tokens (so generation is left-to-right).
- Cross-Attention – attends to the encoder output (aligns input and output).

4️⃣ Output Layer

- Final decoder output → Linear layer → Softmax
- Produces probability distribution over vocabulary for the next token.
