    - What is keys and queries matrix in Attention layer?
        - keys and quires are the mechanism to measure relevance between tokens
        - The purpose of Keys is to provide a way for each token to “advertise” what kind of information it holds.
        - Example: Imagine a cocktail party:
            - Queries are questions you ask (“Who is the subject of this sentence?”)
            - Keys are name tags people wear (“I’m a noun!” or “I’m the verb!”)
            - You compare your query with everyone’s key → decide who to talk to → then get their value (information)
